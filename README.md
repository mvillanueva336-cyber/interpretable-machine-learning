# Interpretable Machine Learning

This project explores ensemble learning and model interpretability techniques using the Boston Housing dataset in Python.

## Project Overview

The Boston Housing dataset was preprocessed and divided into training and testing sets. Multiple tree based regression models were trained and optimized using Grid Search Cross Validation.

The project goes beyond predictive accuracy by exploring feature importance and model interpretability using Partial Dependence Plots (PDP) and Accumulated Local Effects (ALE).

## Models Implemented

* Decision Tree Regression
* Random Forest Regression
* Gradient Boosting Regression

## Hyperparameter Tuning

* Grid Search Cross Validation
* Five Fold Cross Validation

## Model Interpretation Techniques

* Feature Importance
* Partial Dependence Plots (PDP)
* Accumulated Local Effects (ALE)

## Evaluation Metrics

* Root Mean Squared Error (RMSE)
* Mean Absolute Error (MAE)
* R Squared (R²)
